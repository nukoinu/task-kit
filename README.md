# Task-Kit

## Codex 実行例

`.github/prompts/codex-collaboration.prompt.md` を使う場合は、次の入力をそのまま渡します。

```md
依頼名: 用語レビュー基準の整理
目的: レビュー観点を統一し、見落としを減らす
対象成果物: docs/terminology/term-governance.md
成功条件:
- 重大/軽微の判定基準が明文化されている
- 例外ケースの扱いが記載されている
制約/禁止事項:
- 既存の章構成は維持する
- 用語定義の意味変更はしない
参照資料:
- docs/terminology/misuse-log.md
- docs/templates/common-agent-input-template.md
```
