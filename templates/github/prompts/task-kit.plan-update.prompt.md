# /task-kit.plan-update

Task-Kit の計画更新を `task-kit.plan` へ委譲する。

## 入力
- タスクパス（任意。未指定時は `.task-kit/current-task.md` を参照）
- 計画更新内容（必須）
- 制約・完了条件・レビュー要件（任意）
