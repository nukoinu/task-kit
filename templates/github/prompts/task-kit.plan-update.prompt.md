# /task-kit.plan-update

Task-Kit の計画更新を `task-kit.plan` へ委譲する。

実行可能な計画が確定した場合は、現在の会話で実行を続けず、新しいセッションを開始するよう利用者へ案内する。新セッションへそのまま貼り付けられる、自己完結した `実行セッションパッケージ` を一つのコードブロックで提示する。パッケージには `/task-kit.task-execute`、タスクパス、対象ステップ、参照先、制約、受け入れ条件、検証方法、期待する出力形式を含め、会話履歴、未確定事項、秘密情報、他タスクの文脈を含めない。

## 入力
- タスクパス（任意。未指定時は `.task-kit/current-task.md` を参照）
- 計画更新内容（必須）
- 制約・完了条件・レビュー要件（任意）
