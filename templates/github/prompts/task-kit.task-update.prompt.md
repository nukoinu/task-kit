# /task-kit.task-update

目的: タスク定義を更新する。

## 入力
- タスクパス(必須)
- 変更要求(必須)

## 手順
1. タスクパス配下の task.md を確認する。
2. 変更要求を反映し、既存構成を維持して最小差分で更新する。
3. 変更履歴に更新内容を追記する。
4. records/findings.md に変更理由を要約する。

## 出力形式
- 更新対象
- 変更要約
- 影響範囲

## 禁止事項
- 機密情報、資格情報、個人情報を記録しない。
- 成果物本文にタスクキット利用事実を混入しない。
