# /task-kit.plan-update

目的: 実行計画を更新する。

## 入力
- タスクパス(必須)
- 計画更新内容(必須)

## 手順
1. plan.md を読み、未完了ステップと依存関係を確認する。
2. 更新内容を反映し、実施ステップを更新する。
3. 実行前リスクと実行中リスクを整理する。
4. records/scratchpad.md に検討メモを追記する。

## 出力形式
- 更新対象
- 変更要約
- リスク更新

## 禁止事項
- 機密情報、資格情報、個人情報を記録しない。
- 成果物本文にタスクキット利用事実を混入しない。
