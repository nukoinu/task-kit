# Findings

## 1. 記録ルール
- 事実、判断、根拠を分けて記録する。
- 機密情報、資格情報、個人情報は記録しない。
- 確定した事実、判断、懸念と根拠を追記する。追記時は日時を必ず記録する（`YYYY-MM-DD HH:mm`、ローカル時刻）。作業中の仮説や一時メモは scratchpad.md に記録する。
- review agent は新規に特定した事実、欠陥、根拠だけを追記し、既存記録を変更しない。
